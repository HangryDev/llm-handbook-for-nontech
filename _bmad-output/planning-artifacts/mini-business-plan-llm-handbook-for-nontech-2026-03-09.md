## 1. Vision

AI가 빠르게 발전하면서 비전공자들은 무엇을 배워야 하는지 따라가기 어렵다.

> As AI evolves rapidly, non-technical professionals struggle to keep up with what to learn.

**We are building:**

### AI Workflow Operating System for Professionals

| Goal | Description |
|---|---|
| Curate | Select and filter the AI information that matters |
| Build | Create ready-to-use workflows |
| Grow | Develop non-technical users into AI Champions |
| Ecosystem | Build a community-driven AI ecosystem |

## 2. Problem

Current problems in the AI learning market:

| # | Problem | Description |
|---|---|---|
| 1 | Information overload | AI news and tools change too fast |
| 2 | Lack of practicality | Many courses exist but real-world application is hard |
| 3 | No maintenance | Workflows quickly become obsolete |
| 4 | Hard to learn alone | No guided community support |

## 3. Solution

**AI Workflow Subscription** — what we provide:

1. Curated AI updates
