# 1. AI가 제일 잘 하는 업무 시스템

## 개요

AI가 가장 잘 하는 업무를 파악하고 적재적소에 활용하는 방법을 학습합니다.

---

### 데이터 분석

- Excel/CSV 데이터 분석
- 통계 분석
- 시각화
- 인사이트 도출

- Grant 제안서 작성
- 시장 분석

- 진행 상황 추적

### Email Triage

- 이메일 분류
- 우선순위 결정
- 답변 초안 작성
- 후속 조치 관리

## 1.3 직무별 활용 프롬프트

### 마케팅

```
당신은 디지털 마케팅 전문가입니다.
제품: [제품명]
타겟: [타겟 고객]
목표: [마케팅 목표]

위 조건으로 다음을 작성해주세요:
1. SNS 캠페인 기획안
2. 콘텐츠 캘린더
3. 예산 배분안
```

### 세일즈

```
당신은 B2B 세일즈 전문가입니다.
잠재 고객: [회사명/산업]
우리 제품: [제품명과 가치 제안]

이메일 콜드 아웃리치 3가지 버전을 작성해주세요:
1. 가치 중심형
2. 문제 해결형
3. 호기심 유발형
```

### PM (Product Manager)

```
당신은 제품 관리자입니다.
기능: [기능 설명]
목표 사용자: [사용자 페르소나]

다음을 작성해주세요:
1. PRD (Product Requirements Document)
2. 사용자 스토리
3. 성공 지표 (KPI)
```
