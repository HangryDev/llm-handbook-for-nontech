## 0.3 체험: 간단한 앱 만들기

### 미션: "할 일 관리 앱 만들기"

```
1. Lovable에 접속
2. "할 일 관리 앱을 만들어줘" 입력
3. 요구사항 추가:
   - 할 일 추가/삭제/완료
   - 날짜별 필터링
   - 다크 모드
4. 결과 확인
```
