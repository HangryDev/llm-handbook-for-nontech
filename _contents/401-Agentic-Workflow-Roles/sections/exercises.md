## 연습 1: EDD 적용

### 문제

자신의 AI 워크플로우에 EDD를 적용해보세요.

### 단계

1. 평가 지표 설정
2. 테스트 케이스 준비 (5개)
3. 실행 및 결과 기록
4. 분석 및 개선안 도출

### 제출물

```
[워크플로우 이름]

1. 평가 지표
   - 지표 1: [이름, 측정 방법]
   - 지표 2: [이름, 측정 방법]

2. 테스트 케이스
   | 케이스 | 입력 | 예상 출력 |
   |--------|------|-----------|
   | 1 | | |
   | 2 | | |

3. 실행 결과
   | 케이스 | 실제 출력 | 점수 |
   |--------|-----------|------|
   | 1 | | |
   | 2 | | |

4. 분석 및 개선안
   [개선이 필요한 부분과 개선 방법]
```
