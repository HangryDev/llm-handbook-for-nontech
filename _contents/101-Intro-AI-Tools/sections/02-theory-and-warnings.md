# 2. 이론 & 주의사항

## 개요

AI의 기본 작동 원리를 이해하고, 사용 시 주의해야 할 사항을 학습합니다.

---

## 2.1 '생성'의 원리

### 생성형 AI란?

생성형 AI(Generative AI)는 새로운 콘텐츠를 만들어내는 AI입니다. 텍스트, 이미지, 음악, 코드 등 다양한 형태의 콘텐츠를 생성할 수 있습니다.

### 확률적 생성

LLM은 다음에 올 단어를 **확률적으로** 예측하여 텍스트를 생성합니다. 결정론적(똑같은 입력에 항상 똑같은 출력)이 아니라, 매번 다른 결과가 나올 수 있습니다.

<!-- INFORAPHIC PLACEHOLDER: 텍스트 생성 과정 -->
![텍스트 생성 과정](_assets/text-generation-process.png)
*디자인 필요: "The cat sat on the" → 다음 단어 확률 분포 → "mat"(60%), "chair"(25%), "floor"(15%)*

### Temperature

Temperature는 생성의 다양성을 조절하는 파라미터입니다.

- **낮은 temperature (0.1-0.3)**: 더 결정론적, 집중된 출력
- **높은 temperature (0.7-1.0)**: 더 창의적, 다양한 출력

## 2.2 '비결정성' (Non-determinism)

### 같은 입력, 다른 출력?

LLM은 같은 프롬프트를 입력해도 매번 다른 답변을 생성할 수 있습니다. 이를 비결정성이라고 합니다.

<!-- DEMO VIDEO PLACEHOLDER -->
[DEMO VIDEO: 같은 프롬프트, 다른 결과 비교 (1분)]

### 왜 비결정적인가?

- 확률적 샘플링 과정
- Temperature 설정
- Seed 값의 변화

### 비결정성의 장단점

**장점:**
- 창의적인 아이디어 생성
- 여러 버전의 결과물 얻기
- 다양한 관점 탐색

**단점:**
- 재현성 부족
- 품질 편차
- 테스트와 검증 어려움

---

## 2.3 할루시네이션 (Hallucination)

### 할루시네이션이란?

AI가 사실이 아닌 내용을 마치 사실인 것처럼 생성하는 현상입니다. 이를 '환각' 또는 '거짓말'이라고도 합니다.

<!-- INFORAPHIC PLACEHOLDER: 할루시네이션 예시 -->
![할루시네이션 예시](_assets/hallucination-example.png)
*디자인 필요: AI가 실존하지 않는 논문이나 사실을 인용하는 모습*

### 할루시네이션의 원인

1. **확률적 생성**: 다음에 올 말단어를 예측하는 과정에서 오류 발생
2. **학습 데이터의 편향**: 잘못된 정보가 학습 데이터에 포함
3. **압축된 지식**: 방대한 정보를 압축하여 저장하다가 세부 정보 손실
4. **질문의 모호성**: 불명확한 질문에 대해 추측으로 답변

### 할루시네이션 탐지 방법

| 징후 | 설명 |
|------|------|
| 구체적이지 않은 답변 | "아마도", "가능성" 등 모호한 표현 |
| 출처 불명 | 구체적인 출처나 근거 제시 불가 |
| 논리적 모순 | 앞뒤가 맞지 않는 설명 |
| 과도한 확신 | 확실하지 않은 사실을 자신 있게 주장 |
| 최신 정보 부재 | 학습 데이터 기간 이후의 정보 |

### 할루시네이션 방지 전략

1. **검증**: 중요한 정보는 반드시 검증
2. **출처 요청**: "출처를 명시해주세요" 프롬프트 사용
3. **체크리스트**: 사실 여부를 확인하는 체크리스트 활용
4. **적당한 회의성**: AI의 답변을 맹신하지 않기

---

## 2.4 기타 주의사항

### 편향성 (Bias)

학습 데이터에 포함된 편향이 AI의 출력에 반영될 수 있습니다. 성별, 인종, 지역 등 다양한 편향이 존재할 수 있습니다.

### 프라이버시 (Privacy)

민감한 개인정보를 AI에 입력하는 것은 위험할 수 있습니다. 회사 내부 정보, 개인정보 등은 주의해서 다루어야 합니다.

### 보안 (Security)

AI 도구를 통해 생성된 코드나 설정은 보안 검증이 필요할 수 있습니다.

## 요약

- **비결정성**: 같은 입력도 다른 출력 가능
- **할루시네이션**: 사실이 아닌 내용을 생성하는 현상
- **주의사항**: 편향성, 프라이버시, 보안
