## 대전략 이해

### 3 Person Unicorn, Centaur, Cyborg

![AI 협업 모델](_assets/ai-collaboration-models.png)
*디자인 필요: 3 Person Unicorn, Centaur, Cyborg 개념도*

- **3 Person Unicorn**: 한 명이 세 명의 몫을 해내는 방식
- **Centaur**: 사람과 AI가 함께 결정을 내리는 협업 방식
- **Cyborg**: AI를 도구로 활용하여 업무 수행

오늘의 기술 최전선에서는 AI가 사람을 능가하는 분야와 사람이 우위인 분야가 공존합니다.

## 이 핸드북을 시작하기 전에

### 전제 조건

- 컴퓨터 기본 사용 능력 (인터넷 브라우저 사용, 파일 관리 등)
- AI 도구에 대한 호기심과 열린 마음
- 영어 읽기 능력 (기술 문서를 위해 권장)

### 필요한 도구

- 인터넷이 연결된 컴퓨터
- Google 계정 (Google AI Tools 사용)
- (선택) OpenAI API Key, Anthropic API Key

### 학습 방법

1. 각 섹션의 설명을 읽습니다
2. 제공된 예시를 따라해봅니다
3. 연습 문제를 직접 풀어봅니다
4. 자신의 프로젝트에 응용해봅니다
