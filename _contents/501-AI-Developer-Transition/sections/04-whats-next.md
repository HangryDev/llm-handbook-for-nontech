## 4.2 기존에 존재하던 것들을 엮기

혁신은 새로운 것이 아니라 기존 것들의 조합입니다.

### 조합의 예시

- LLM + Database = RAG
- LLM + Tools = Agent
- LLM + UI = Chatbot
