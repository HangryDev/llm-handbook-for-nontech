# 2. Vibe Coding

## 개요

AI 코딩 도구를 적극적으로 활용하는 방법을 배웁니다.

---

## 2.1 Lovable

웹 앱을 빠르게 만드는 AI 도구입니다.

<!-- DEMO VIDEO PLACEHOLDER -->
[DEMO VIDEO: Lovable 심화 사용법 (4분)]

### Lovable 활용 전략

1. **프로토타이핑**
   - 아이디어 빠르게 구현

2. **UI 디자인**
   - 자연어로 디자인 요청

3. **배포**
   - 클릭 몇 번으로 배포

## 2.2 Antigravity

전체 스택 앱을 만드는 AI 도구입니다.

### Antigravity 특징

- 프론트엔드 + 백엔드
- 데이터베이스 포함
- 실시간 협업

## 2.3 Cursor

AI 보조 코드 에디터입니다.

### Cursor 기능

| 기능 | 설명 |
|------|------|
| Chat | 코드 질문 및 생성 |
| Composer | 자동 코드 완성 |
| Cmd+K | 코드 수정 |
| Cmd+L | 코드 설명 |

---

## 2.4 Vibe Coding 모범 사례

### 1. 명확한 요청

```
[나쁜 예]
"로그인 만들어줘"

[좋은 예]
"이메일과 비밀번호로 로그인하는 기능을 만들어줘.
Firebase Authentication을 사용하고,
로그인 성공 후 대시보드로 이동해줘."
```

### 2. 단계적 개발

```
Step 1: 기본 UI
Step 2: 상태 관리
Step 3: API 연결
Step 4: 에러 처리
```

### 3. 점진적 복잡성

간단한 것부터 시작해서 점차 복잡하게 만드세요.

---

## 요약

- **Antigravity**: 전체 스택 앱

- **모범 사례**: 명확한 요청, 단계적 개발, 점진적 복잡성
